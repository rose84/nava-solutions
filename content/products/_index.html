Cloud Migration

Simplify Your Journey To The Cloud With Nava Certus!

Nava Certus allows cloud storage customers to migrate windows file systems to the cloud reliably and quickly. While it is easy enough to be used by any user, it was designed with MSPs in mind to accomplish very large data migrations tasks with ease.

Features

  • Migrate from a Windows file server, Google Drive and Dropbox to Google Apps.
  • Migrate from a Windows file server, Google Drive and Dropbox to Amazon S3.
  • Migrate from a Windows file server, Google Drive and Dropbox to Dropbox.
  • Migrate from Google Drive and Dropbox to Windows file server.
  • Maintain File Permissions.
  • Two modes of operation for migrations to Google Drive:
  • One-to-one
  • Distribute files to owners
  • Migration Status Notifications
  • Reporting
  • Continuous Migration mode
